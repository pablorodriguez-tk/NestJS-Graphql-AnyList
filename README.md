# Dev

1. Clonar el proyecto
2. Copiar el `.env.example` y renombrar a `.env`
3. Ejecutar

```
npm install
```

4. Levantar la imagen (Docker Desktop debe estar instalado)

```
docker-compose up -d
```

5. Ejecutar el backend de Nest

```
npm run start:dev
```
